# smart-image-cleaner
Smart Image Cleaner

一个基于 图像内容分析 的照片整理工具，用于：

🔁 查找重复图片（基于感知哈希 pHash）

📸 检测模糊图片（基于清晰度算法）

适用于大规模相册整理（实测支持 10,000+ 张图片）。

🧨 为什么要做这个工具？

在整理照片时，如果使用“按拍摄时间重命名”的工具，很容易出现：

同一张照片被复制多份

元数据被修改

文件大小略有差异

文件名不同但画面完全一致

此时：

❌ MD5 / SHA256 无法识别重复

❌ 仅按文件大小比较无效

❌ 仅按文件名比较无效

本工具通过 图像视觉特征 判断是否重复，而不是比较文件字节。

✨ 功能特性

🔍 基于 pHash 的重复图片检测（视觉级去重）

📸 模糊图片检测（拉普拉斯方差算法）

🧠 自动选择最佳版本（分辨率 / 体积 / 清晰度综合评分）

📁 支持递归扫描文件夹

🛡 默认安全模式（预览不删除）

🚀 支持上万张图片批量处理
